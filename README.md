# SuperStore Sales Dashboard - Excel Project

## Overview
The SuperStore Sales Dashboard is an interactive Excel project designed to analyze sales performance, profit, discounts, and customer order trends. This dashboard helps businesses make data-driven decisions by visualizing key performance metrics.

## Objectives
- Analyze sales and profit performance
- Track order trends and customer insights
- Identify high-performing categories and sub-categories
- Monitor discounts and delivery performance

## Features
- Interactive Dashboard using Pivot Tables and Pivot Charts
- Dynamic filtering with Slicers
- KPI Metrics for Sales, Profit, Orders, and Discounts
- Category-wise and Region-wise Analysis
- Delivery Duration Analysis
- Data-driven insights for decision-making

## Excel Functions Used
- `SUM()`
- `AVERAGE()`
- `COUNTIF()`
- `COUNTA()`
- `GETPIVOTDATA()`
- `IF()`
- Date Difference Formula

## Aggregate Functions Used
- Sum of Sales
- Sum of Profit
- Count of Orders
- Average Discount
- Category and Sub-category Aggregation

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Dashboard Design
- Data Analysis

## Key Insights
- Identified top-performing product categories
- Analyzed profit trends across regions
- Tracked discount impact on sales performance
- Measured delivery duration using order and ship dates

## Project Outcome
Created an interactive dashboard that simplifies business reporting and enables quick analysis of sales and operational performance.
